ℹ️ About This Dashboard

Real-time health monitoring visualization system

  • Receives live sensor data via WebSocket connection
  • Updates metrics every second
  • Charts historical data for trend analysis
  • Alerts when readings fall outside safe ranges
  • Currently running in simulation mode with realistic data

Safe Ranges:

  • Heart Rate: 60-110 BPM
  • Oxygen Saturation: ≥ 90%

⚠️ Educational prototype only. Not for medical use. See project documentation for safety information.
