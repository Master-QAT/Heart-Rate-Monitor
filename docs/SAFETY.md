# Safety & Medical Disclaimer

## ⚠️ CRITICAL NOTICE

**THIS IS NOT A MEDICAL DEVICE**

This Heart Rate Monitor is an **educational prototype** designed for learning embedded systems and sensor technology. It should **NEVER** be used for:
- Medical diagnosis
- Patient monitoring
- Clinical decision-making
- Healthcare applications without professional supervision

---

## Limitations

### Accuracy Limitations
- Expected accuracy: ±5-10% (compared to clinical devices)
- Clinical pulse oximeters: ±2-3% accuracy
- This device: Educational demonstration only

### Physical Limitations
- Environmental factors affect readings
- Skin tone impacts sensor performance
- Nail polish or artificial nails reduce accuracy
- Motion artifacts cause incorrect measurements
- Cold hands reduce signal quality

### Hardware Limitations
- Not calibrated to medical standards
- No FDA or CE certification
- No ongoing quality assurance
- No professional testing protocols

## Safety Guidelines

### Hardware Safety
1. **Electrical Safety**
   - Use 3.3V / 5V logic only
   - Do not exceed voltage specifications
   - Properly ground all components
   - Use proper USB power supply

2. **Sensor Safety**
   - Do not apply excessive pressure to finger
   - Do not modify sensor connections
   - Keep sensor clean
   - Avoid contamination

3. **Environmental Safety**
   - Use in dry environment
   - Avoid water/moisture contact
   - Keep away from magnetic fields
   - Use proper ventilation

### User Safety
1. Remove all jewelry before use
2. Use clean hands
3. Don't use if skin is broken
4. Stop if uncomfortable
5. Consult doctor if concerned
